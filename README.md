# 🧠 Online Quiz Application

An interactive **web-based quiz application** built using **HTML, CSS, and JavaScript**.
The app presents multiple-choice questions, checks answers instantly, and displays the final score to the user.

---

## 🚀 Live Demo

👉 https://riaz1407.github.io/quiz_app/

---

## 📌 Features

* Multiple choice questions
* Real-time answer validation
* Automatic score calculation
* Next question navigation
* Final score display
* Clean and responsive interface

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (DOM Manipulation & Event Handling)

---

## ⚙️ How It Works

1. User clicks **Start Quiz**
2. Questions load dynamically
3. User selects an option
4. App checks the answer instantly
5. Score updates automatically
6. Final result appears at the end
